### 🔧 技术实现

#### 核心流程
```
商品标题 → 文本清洗 → jieba分词 → 词性标注 → 多重过滤 → 标签提取 → 数据库存储
```

#### 多重过滤机制
1. **长度过滤**: 过滤长度<2的词汇
2. **停用词过滤**: 包含10大类停用词（常用词、商品词、营销词等）
3. **词性过滤**: 只保留n、v、a、eng等有意义词性
4. **数字过滤**: 移除纯数字，保留型号等
